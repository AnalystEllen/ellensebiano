## 🚨 911 Call Monitoring System

A structured **database and dashboard system designed to monitor and track emergency calls received by the city's 911 services**.

The system organizes call information and supports monitoring of **city emergency resources such as ambulances, fire trucks, and response units**.

**Key Contributions**

- Designed a structured database for recording 911 call information  
- Organized fields for tracking incidents, response type, and location  
- Monitored deployment of city emergency resources  
- Developed a **dashboard using Google Sheets** for monitoring emergency call activity  

**Key Features**

- Call incident tracking  
- Emergency response categorization  
- Resource monitoring (ambulances, fire trucks, etc.)  
- Dashboard visualization for call trends and response data  

**Tools Used**

- Google Sheets  
- Excel
- Looker Studio

## 🎮 BIGHANI: Philippine Mythology NPC Simulation

A university capstone project that preserves **Philippine lower mythological creatures** through a **2D RPG simulation**.

The project includes a **database of over 400 mythological creatures**, used to simulate behaviors through **NPC finite state machine models**.

**Key Contributions**

- Created and organized the mythology creature database  
- Ensured accurate classification of mythological entities  
- Provided structured data for NPC behavior simulation
